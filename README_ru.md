# 2048 Игра

Классическая реализация головоломки 2048, построенная на чистом JavaScript. Двигайте пронумерованные плитки по сетке, чтобы объединить их и создать плитку с числом 2048.

## 🎮 О Игре

Это браузерная реализация популярной игры 2048. Цель — двигать пронумерованные плитки по сетке 4×4, чтобы объединить их и создать плитку с числом 2048. Когда две плитки с одинаковым числом соприкасаются, они сливаются в одну с их суммой.

## ✨ Возможности

- **Чистый Vanilla JavaScript** — Без фреймворков и библиотек
- **Логика Слияния Плиток** — Плавная механика объединения плиток
- **Управление Состоянием на Основе Сетки** — Чистая обработка и обновление состояния
- **Управление с Клавиатуры** — Стрелки для движения плиток
- **Адаптивный Дизайн** — Работает на разных размерах экрана
- **Отслеживание Очков** — Следите за текущим счётом
- **Определение Конца Игры** — Автоматическое определение, когда ходы невозможны

## 🛠️ Технологии

- **HTML5** — Семантическая разметка
- **CSS3 / SCSS** — Стилизация с препроцессором
- **Vanilla JavaScript** — Игровая логика и манипуляция DOM
- **ES6 Модули** — Модульная структура кода

## 🚀 Начало Работы

### Требования

- Современный браузер (Chrome, Firefox, Safari, Edge)
- (Опционально) Локальный веб-сервер для разработки

### Установка

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd 2048
```

2. Откройте `index.html` в браузере или используйте локальный сервер:
```bash
# Используя Python
python -m http.server 8000

# Используя Node.js http-server
npx http-server
```

3. Перейдите на `http://localhost:8000` в браузере

## 🎯 Как Играть

1. Используйте **стрелки** (↑ ↓ ← →) для движения плиток
2. Когда две плитки с одинаковым числом соприкасаются, они **сливаются в одну**
3. После каждого хода появляется новая плитка случайным образом
4. Цель — создать плитку с числом **2048**
5. Игра заканчивается, когда больше нет возможных ходов

Если вы изменяете SCSS файлы, скомпилируйте их в CSS:
